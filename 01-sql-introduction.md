# 1 - Intro

## 📌 What is SQL?

- **SQL (Structured Query Language)** is used to interact with databases  
- It helps us perform **CRUD operations**:

  - **C** → Create (Insert data)  
  - **R** → Read (Fetch data)  
  - **U** → Update (Modify data)  
  - **D** → Delete (Remove data)  

💡 **Shortcut memory tip:**  
👉 *“SQL = Talk to Database (Create, Read, Update, Delete)”*

---

## 📌 What is DBMS?

- **DBMS (Database Management System)** is software used to:
  - Store data  
  - Manage data  
  - Retrieve data efficiently  

💡 **Examples:** MySQL, Oracle, SQL Server  

---

## 📌 Types of Databases

### 1️⃣ Relational Database (RDB)

- Data is stored in **table format (rows & columns)**  
- Tables are related using **keys (Primary Key, Foreign Key)**  
- Structured and organized  

💡 **Example:**

| ID | Name | Age |
|----|------|-----|
| 1  | John | 25  |
| 2  | Jane | 30  |

---

### 2️⃣ Non-Relational Database (NoSQL)

- Data is **NOT stored in tables**  
- Uses flexible formats like:
  - JSON  
  - Graph  
  - Key-Value pairs  

💡 **Used when:**
- Data is unstructured  
- High scalability is needed  

---

## 📌 What is RDBMS?

- **RDBMS (Relational Database Management System)**:
  - Software used to manage **relational databases**  
  - Works with **tables and relationships**  

💡 **Examples:**
- MySQL  
- Oracle Database  
- Microsoft SQL Server  

---

## 📌 What are we using?

- We are learning SQL using **MySQL**  
- It is one of the most popular **RDBMS tools**  

💡 **Why MySQL?**
- Easy to learn  
- Widely used in web development  
- Open-source  

---

## ⚡ Final Memory Summary 

- SQL → Used for CRUD operations  
- DBMS → Software to manage databases  
- Databases → 2 types:
  - Relational (tables)  
  - Non-relational (JSON, graph, etc.)  
- RDBMS → Manages relational databases  
- Tool used → MySQL
